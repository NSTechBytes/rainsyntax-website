Getting Started

RainSyntax is a powerful VS Code extension designed to enhance your Rainmeter development experience.

Quick Start

  1. Install RainSyntax from the VS Code Marketplace
  2. Open a Rainmeter .ini file in VS Code
  3. Start coding with enhanced features

Installation

Follow these steps to install RainSyntax in VS Code:

  1. Open VS Code.
  2. Go to the Extensions Marketplace (Ctrl+Shift+X).
  3. Search for "RainSyntax" and click Install.
  4. Reload VS Code if required.
  5. Start using the extension on Rainmeter files.

Features

  • Syntax Highlighting: Advanced highlighting for Rainmeter syntax.
  • Auto-Refresh Skin: Automatically refreshes Rainmeter skins while coding.
  • GUI Customization: Allows users to modify UI settings.
  • Hover Information: Displays helpful tooltips on Rainmeter elements.
  • Color Picker: Integrated color picker for easy color selection.
  • Code Formatting: Ensures clean and consistent code styling.

Configuration

Customize RainSyntax settings in VS Code by modifying the settings.json file:

{
  "rainsyntax.autoRefresh": true,
  "rainsyntax.colorPicker": true,
  "rainsyntax.formatOnSave": true
}

These options allow you to enable or disable features according to your workflow.

API Reference

RainSyntax provides an API for advanced integration. Below are some key commands:

  • rainSyntax.toggleAutoRefresh - Toggles AutoRefresh for Rainmeter skins.
  • rainSyntax.changeRainmeterPath - Changes the Rainmeter executable path.
  • rainSyntax.changeRefreshMode - Switches between refreshing all skins or only the active skin.
  • rainSyntax.openSettings - Opens RainSyntax settings in VS Code.

Configuration Settings

Customize RainSyntax behavior using these settings in settings.json:

  • rainSyntax.autoRefreshOnSave (boolean, default: true) - Automatically refresh Rainmeter skins when saving an INI or INC file.
  • rainSyntax.refreshMode (string, default: "all") - Choose between refreshing all skins or only the specific skin being edited.
  • rainSyntax.rainmeterPath (string, default: "C:\Program Files\Rainmeter\Rainmeter.exe") - Path to the Rainmeter executable.
